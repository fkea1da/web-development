Отчет по работе с Git      

Тема: Работа с Git

**Цель работы**
Изучить базовые команды Git, научиться инициализировать репозиторий, создавать коммиты, работать с ветками, выполнять push в удаленный репозиторий, а также освоить корректное удаление файлов с учетом особенностей Git и файловой системы.

**Инициализация репозитория и создание файлов**
git init
touch readme.txt "readme after.txt" "readme now.txt"
Файлы были созданы в рабочей директории

**Проверка состояния**
git status

**Добавление файла в Git и коммит**
git add readme.txt
git commit -m "...."

**Работа с ветками**
git branch clean_up
git checkout clean_up

** Подключение удаленного репозитория и push**
.....
git push -u origin main

**Ключевые выводы(принцип работы)**
git add → git commit → git push
